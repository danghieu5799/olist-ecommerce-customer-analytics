# Python Analysis

This folder contains the Jupyter Notebook used for data cleaning, exploratory data analysis (EDA), and customer analytics for the Olist E-commerce project.

## Notebook

### 01_eda_olist.ipynb

Main tasks performed:

* Data loading and inspection
* Data cleaning and preprocessing
* Exploratory Data Analysis (EDA)
* Revenue analysis
* Customer behavior analysis
* Customer segmentation (RFM)
* Repeat customer analysis
* Cohort analysis
* Business insight generation

## Libraries Used

* Pandas
* NumPy
* Matplotlib
* Seaborn

## Output

The analysis results were used to support:

* Customer segmentation
* Retention analysis
* Revenue performance evaluation
* Power BI dashboard development
* Business recommendations
