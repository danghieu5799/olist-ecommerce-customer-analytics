# SQL Scripts

This folder contains SQL scripts used to build the Data Warehouse for the Olist E-commerce Analytics project.

## Files

### 01_staging_tables.sql

Creates staging tables to store and prepare raw data before transformation.

### 02_dim_fact_tables.sql

Creates dimension tables and fact tables following the Star Schema design.

## Database

SQL Server

## Purpose

Support ETL processing and analytical reporting through a structured Data Warehouse model.
