# Power BI Dashboard

## Overview

This folder contains the Power BI dashboard developed for the Olist E-commerce Customer Analytics project.

The dashboard was built to support business monitoring, customer analysis, and revenue performance tracking.

---

## Dashboard Features

### Executive Overview

* Revenue KPI
* Customer KPI
* Order KPI

### Customer Analytics

* Customer Segmentation
* RFM Analysis
* Repeat Customer Analysis

### Revenue Analytics

* Revenue Trends
* Sales Performance

### Retention Analytics

* Cohort Analysis
* Customer Retention Trends

---

## Technologies Used

* Power BI Desktop
* Power Query
* DAX
* Data Modeling

---

## Output

The dashboard supports business decision-making by providing interactive visualizations and actionable insights.
