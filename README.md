# Olist E-commerce Customer Analytics

## Project Overview
This project analyzes customer purchasing behavior, retention performance, and revenue trends using the Olist E-commerce dataset.
The objective is to identify high-value customer segments, evaluate customer retention, and generate actionable business insights to support business growth.

## Business Problem
An e-commerce platform aims to understand customer purchasing behavior, repeat purchase rates, and the key factors influencing revenue in order to improve business performance and drive sustainable growth.

## Dataset
Source: Olist E-commerce Dataset
Customers: ~96,000
Orders: ~99,000
Products: ~32,000
Order Items: ~112,000
